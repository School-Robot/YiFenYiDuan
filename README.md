# YiFenYiDuan
基于百度API及掌上高考数据的一分一段表查询工具，可自行修改用于批量处理数据

# 测试环境
Windows 11

Python 3.12

# 使用方法
1. 使用`pip`安装`requests`库
2. 使用`python3 yifenyiduan.py`运行脚本
3. 根据提示输入省份、批次、类别
4. 等待脚本获取数据，直到出现分数提示
5. 输入分数即可出现分数范围及排名范围，可循环输入
6. 输入`exit`退出
